在macOS 10.11+下安装配置Jekyll

2013-04-30 08:00

在macOS 10.11+下安装配置Jekyll

  1. 确保系统已经有环境：Ruby(v1.9.3+),RubyGems,NodeJS,Python 2.7 ;
  2. 安装Xcode Command-Line Tools：xcode-select --install；
  3. 安装Jekyll:sudo gem install -n /usr/local/bin jekyll，检查版本：jekll -v；
  4. 安装Github Pages插件：sudo gem install -n /usr/local/bin github-pages；
  5. 查看Gems列表：gem list；
  6. 启动Jekyll服务：jekyll serve；

参考：Jekyll官网
